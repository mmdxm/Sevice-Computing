# CLI 命令行实用程序开发实战 - Agenda

### 功能

在这次作业中，我实现了register和login两个功能

register：用于注册一个用户账号，用法

$Agenda register -u username -p password -e email -o phone

login:用于登录一个账号，用法

$Agenda login -u name -p password
